Importing the ‘~’ delimited text file into Excel.

a. Right click on the attached ~delimited .txt file and save the text file on your PC or flash disk.
b.Open Excel.
c.Click on Open and open the .txt file. Change the “Files of Type” to Text files ( *.prn; *.txt; *.csv) if needed.
d.Excel will take you through a Text Import Wizard of 3 steps.
 Step 1:
 Choose “Delimited” as the Original data type.
 Click on Next.
 Step 2:
 Choose “Other” as the Delimiter and type a ~ in the delimiter field next to “Other”.
 Click on Next.
 Step 3:
 Leave all the column formats on the default of “General”.
 Click on Finish.
e.The file will open. Format the file as preferred.
f.Save the file as an Excell Workbook ( .xlsx or .xls).
