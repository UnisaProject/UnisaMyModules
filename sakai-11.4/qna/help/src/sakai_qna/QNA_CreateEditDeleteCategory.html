Create / edit / delete a category

Create a category

To create a category:

  • On the default page, click the 'Add a category' link
  • Fill in the category name
  • If you wish to create additional categories, click 'Add another category'
  • When you're finished click 'Save'

Edit a category

Note: this feature is only applicable for those with update rights
To edit a category name:

  • On the default page, click the 'Edit 'link alongside the category name
  • Edit the text in the field provided
  • When you're finished click 'Save'

OR

  • Click the 'Organise' link
  • Click the edit icon on the far right of the appropriate category to edit the category name
  • When you're finished click 'Save'

Delete a category

Note: this feature is only applicable for those with update rights
Warning: When deleting a category, all questions and answers associated with that category will also be removed

To remove a category:

  • On the default page, check the box in the Remove column
  • Click the 'Update' button at the bottom of the list

OR

  • Click the 'Organise' link
  • Click the delete icon on the far right of the appropriate category to remove it.

Note: The 'New Questions' category cannot be deleted. It is generated automatically when moderation is switched on.
