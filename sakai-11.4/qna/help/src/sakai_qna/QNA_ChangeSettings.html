Change settings

Note: this feature is only applicable for those with update rights

The Questions and Answers tool has a number of settings that can be adjusted to control:

  • Moderation
  • Anonymity
  • Notifications
  • Default views

To change the settings, click on 'Options'

  • Moderation
    If you select this box, questions will be placed in a hidden category called 'New Questions' from where they can be replied to privately, published in another category, or deleted.
  • Anonymity
    If you select this box, participants will be able to ask questions anonymously
  • Notifications
    If you select this box, notifications wiLl be sent to the specified recipient(s) when new questions are added. Choose from the official site contact person, specific email address(es) or everyone with update rights
  • Default views
    Use these options to show questions by category or according to popularity (hits)

When you are finished, click 'Save'
