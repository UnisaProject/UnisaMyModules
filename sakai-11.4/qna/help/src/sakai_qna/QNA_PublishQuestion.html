Publish a question

Note: this feature is only applicable for those with update rights

If moderation is switched on you will need to view questions in the 'New Questions' folder and then publish them in another category before they can be viewed by site members.

To publish a question:

  • In the 'New Questions' or 'Private Replies' category, click on the question you want to publish
  • Click 'Publish question'
  • Select the category you want to place it in or create a new category
  • If you would like to answer the question at this point, do so in the field provided. Note: If you have already answered this question privately, you will see your original text in the answer field, which you can edit
  • When you're done, click 'Publish'
